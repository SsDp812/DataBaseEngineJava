# Simle engine for simulating sql dataBase.

## About db:
 All tables saved in txt files, but when program is working
 tables are into List<Map<String,Object>> data structure.
 
## Base Types:
1. Long
2. Boolean
3. String

## Operations
1. Select
2. Insert
3. Update
4. Delete

## Filters (where)
1. like
2. ilike
3. >
4. <
5. >=
6. <=
